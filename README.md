# GoBarber-web

## Getting Started

Clone the repository.

```sh
git clone https://github.com/guilhermeabel99/
```

`cd` into the directory.

```sh
cd gobarber-web
```

Install the project dependencies:

```sh
yarn

# or

npm install
```

Start the development server:

```sh
yarn start

# or

npm run start
```
